# 📊 Sistema de Información

## 🏪 Caso: Tienda de ropa
Una tienda de ropa registraba manualmente sus ventas, inventarios y facturas en una libreta, lo que provocaba errores en las cuentas y dificultad para saber cuántos productos quedaban disponibles. Recientemente implementaron un sistema computacional que integra el registro de ventas, el control automático de inventario y la generación digital de facturas.

Ahora, cuando el cajero realiza una venta, los datos se ingresan en el sistema, este procesa la información automáticamente, descuenta los productos del inventario y genera un reporte de ingresos diarios. Gracias a esta tecnología, la tienda puede obtener información más precisa y en tiempo real, facilitando la toma de decisiones como reabastecer mercancía o lanzar promociones.
